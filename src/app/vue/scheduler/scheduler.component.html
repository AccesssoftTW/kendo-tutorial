事前準備

這邊假設已經準備好 Vue 開發環境 3.x版以上
1. vue create kendo-vue-scheduler
2. cd kendo-vue-scheduler
3. npm install --save @progress/kendo-ui
4. npm install --save @progress/kendo-theme-default
5. npm install --save @progress/kendo-scheduler-vue-wrapper
6. 請先下載完整範例程式碼 Github

選擇要呈現樣式

- 它提供了以不同的視圖呈現計劃的事件的選項-作為一天，一整週或一個月，或作為需要完成的任務列表。
- 它也提供了鍵盤指令，更多說明請參考這裡
- 你可以自由設定初始時間、工作時段更可以匯出PDF，下面將為您說明Kendo-Scheduler 裡常用的屬性。
- 更多說明請參考這裡

Kendo-Scheduler 裡常用的屬性

- :toolbar="['pdf']"、:pdf-file-name 設定匯出PDF按鈕以及匯出PDF的檔名，更多詳細說明請參考這裡
- :start-time、:end-time 設定行事曆開始和結束時間
- :timezone 設定時區，另附上時區對照表
- :selectable="true" 設定一次選擇多個時段並可以同時移動他們，更多詳細說明請參考這裡

Kendo-Scheduler 裡常用的方法

- @change當用戶要選擇時段時觸發此事件
- @edit當用戶要編輯既有行程時觸發此事件
- @add當用戶要添加新行程時觸發此事件
- @cancel當用戶要取消行程時觸發此事件
- @move當用戶要移動既有行程時觸發此事件
- @save當用戶要儲存行程時觸發此事件
- @resize當用戶要拖拉調整行程時間時觸發此事件
- @navigate當用戶要改選日期或週期時觸發此事件
- 更多方法說明請參考這裡
