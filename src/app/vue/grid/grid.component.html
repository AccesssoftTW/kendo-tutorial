事前準備

這邊假設已經準備好 Vue 開發環境 3.x版以上
1. vue create kendo-vue-scheduler
2. cd kendo-vue-scheduler
3. npm install --save @progress/kendo-ui
4. npm install --save @progress/kendo-theme-default
5. npm install --save @progress/kendo-grid-vue-wrapper
6. 請先下載完整範例程式碼 Github

- 可簡單設定參數來匯出Excel、PDF
