事前準備

這邊假設已經準備好 Vue 開發環境 3.x版以上
1. vue create kendo-vue-chart
2. cd kendo-vue-chart
3. npm install --save @progress/kendo-ui
4. npm install --save @progress/kendo-theme-default
5. npm install --save @progress/kendo-charts-vue-wrapper
6. 請先下載完整範例程式碼 Github

Kendo-Chart 裡常用的屬性

- :category-axis 對分類軸線進行設定，比如單位、顏色、最大值、最小值等等，更多詳細說明請參考這裡
- :value-axis 對分類軸的所要呈現的值進行設定，更多詳細說明請參考這裡
- :legend 對圖例的顯示，設定位置、高度、寬度等等，更多詳細說明請參考這裡
- :series 對圖表系列類型由類型字段的值判斷，如果缺少類型值會直接使用默認值，更多詳細說明請參考這裡
- :title 對圖表標題的文字設定，更多詳細說明請參考這裡
- :tooltip 對圖表工具提示的設定，更多詳細說明請參考這裡
- :chart-area 對圖表區域設定，更多詳細說明請參考這裡
- :theme圖表主題設定，例如：sass、black、blueopal、bootstrap、default...
- 更多可以參考這裡
