我們大致上需要下面三個檔案

1. jszip.min.js - Kendo提供
2. products.js - 假資料
3. gird.js - 實作程式

然後我們可以參考下方範例，開始我們的演示

Kendo Grid有相當多設定，我們會先從Datasource這一個部份開始看起

1. data - 資料來源

這邊是存放Grid需呈現資料的地方，或是未來也可以提供API路由(參考連結設定)

2. pageSize

設定一頁要顯示的筆數

3. schema - 定義資料格式

這邊可以針對資料類型(bool、string、date)進行定義，讓套件可以自動幫你做對應類型的處理，也可以設定資料驗證

  • 相關連結：
  • Local Data
  • Remote Data

接下來我們進行Columns定義

1. field

定義要顯示Datasource中的哪一個欄位

2. title

如果不想要顯示資料欄位名稱，可以定義要顯示在畫面上的名稱

3. format

如果有數字、日期格式，可以定義呈現的格式

4. command

需要有編輯功能，需要用此屬性增加更新、刪除鈕

更多的功能設定我們可以參考下方的連結

接下來我們可以針對整個Grid做進一步的設定

toolbar

可以設定要顯示在Grid上方的功能鈕 (新增、編輯、儲存、取消、Excel、PDF)

editable

設定是否開放編輯，也可以在更進階設定批次修改

pageable

設定顯示分頁，或是讓使用者自行輸入分頁筆數
