Step 4 : 選擇您要下載的格式

我們可以直接點選 Latest Public Version 按鈕

Step 5 : 說明

下載後，會得到如圖中的一包程式碼
apptemplates - 可以提供快速建立一個專案版型的範例，或是當成參考
examples - 可以查看每個元件的範例，加快上手的時間
typescript - kendo 提供的 typescript 定義檔
src - 用戶可以在此建立一些測試案例

styles 的部份，可以選擇想要的主題包，以 kendo.default.min.css 為例
- kendo.common.min.css [基礎css]
- kendo.default.min.css [kendo 預設的主題包]
- kendo.default.mobile.min.css [針對不同裝置的預設主題包]
- 某些功能會需要多載入其他的 css 主題包

這邊建議可以將 styles 中的程式包先全部放入到自己專案中，並依需求載入需要的主題包
因為主題包中，會相依 styles 中的圖檔、字體檔等等，例如 loading 效果。

js 的部份
- jquery.min.js
- kendo.all.min.js [kendo 全部的功能]
- 某些功能會需要多載入其他的 js

這邊建議可以依專案需求，看是否要載入整包 js 檔，但可能會增加前端程式包的大小
或是根據使用到的功能，依序載入需要用到的 js 檔，讓前端專案的大小得到充份的利用
例如：想要使用 Autocomplete 的功能，即載入 kendo.autocomplete.min.js
