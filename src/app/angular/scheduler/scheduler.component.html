這邊假設已經準備好 Angular 開發環境 v6以上
1. ng new kendo-angular-scheduler
2. cd kendo-angular-scheduler
3. ng add @progress/kendo-angular-scheduler

完整代碼

呈現格式

- 如下圖中，加入對應格式 Scheduler 的 Component 即可以啟用此功能
- 更多說明，請參考 這裡

啟用編輯[預設]

- 先建立一個 FormGroup，假設是 createFormGroup，寫法如下
- [kendoSchedulerReactiveEditing]="createFormGroup"

其他相關操作

- 設定初始日期 [selectedDate]="selectedDate"
- 提供 kendoSchedulerBinding 現有的行事歷資料
- 資料格式如下方圖示

如果要提供客製化的資料處理邏輯給Kendo UI的做法是什麼呢?

- 建立自己的 service 實作邏輯，並繼承 Kendo UI 提供的 BaseEditService
- 需要將提供給 Kendo UI 的參數調整成以下
- [kendoSchedulerBinding]="editService.events | async"
- [kendoSchedulerReactiveEditing]="createFormGroup"
- [modelFields]="editService.fields"
- [editService]="editService"
- 完整範例代碼可以參考 這裡
