事前準備

這邊假設已經準備好 Angular 開發環境 v6以上
1. ng new kendo-angular-scheduler
2. cd kendo-angular-scheduler
3. ng add @progress/kendo-angular-scheduler
4. 請先下載完整範例程式碼 Github

選擇要呈現樣式

- 可以同時啟用不同的呈現樣式
- 設定初始日期 [selectedDate]="selectedDate"
- 你可以加入 startTime="08:00"、endTime="18:00" 設定工作時段
- 更多說明，請參考 這裡

如何啟用編輯

- 需要提供 Scheduler Form 驗證，所以要建立一個 FormGroup，並定義好我們的資料驗證格式
- 建立 Service 處理 Scheduler 的事件，以便我們處理資料邏輯，且要注意繼承 BaseEditService
- 需要定義好自己的行事歷資料欄位與 SchedulerModelFields 的對應關係，以利 Scheduler 識別我們的資料類型
- 提供 loaging 控制參數，讓 Scheduler 可以有處理事件的動畫效果
- 將上述的程式碼準備完成，並提供給 Scheduler
- 更多的應用，可以參考 這裡

時區該怎麼設定

- 先引入 Kendo 的時區包
- 設定 Scheduler 我們要固定時區

如果想要客製 Scheduler 上的事件樣式的話，該怎麼做

- 假設我們想要改變 "整天(AllDay)" 的事件樣式
- 在 kendo-scheduler 這個 component 中先引入 ng-template
- 引入 kendoSchedulerAllDayEventTemplate 這個 directive (不同的事件會有不同的directive)
- let-event="event" 把 Scheduler 的事件參數接出來
- 在 ng-template 中加入我們想要的事件樣式
- 更多的 template 可以參考 這裡
