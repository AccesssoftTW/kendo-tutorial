事前準備

這邊假設已經準備好 Angular 開發環境 v6以上
1. ng new kendo-angular-inputs
2. cd kendo-angular-inputs
3. ng add @progress/kendo-angular-inputs
4. 請先下載完整範例程式碼 Github

文字輸入 - Textbox

- 設定文字框上方的標題文字 floatingLabel="First name"
- 更多其他的設定可以參考 這裡

數字 - Numeric Textbox

- 設定每次遞增/減的間隔 [step]="5"
- 預設會顯示遞增/減的按鈕，也可以將其關閉 [spinners]="false"
- 預設的數字格式為 n2，也可以根據需求顯示其他格式 [format]="'c0'"，更多格式設定可以參考 這裡
- 更多其他的設定可以參考 這裡

遮罩 - MaskedTextbox

0 —必填數值 (0-9).
9 —必填數值 (0-9) 或空白.
# —必填數值 (0-9) 或空白、符號 (+、-).
L —必填文字 (a-Z).
? —必填文字 (a-Z) or 或空白.
A —必填字母、數字 (0-9, a-Z).
a —必填字母、數字 (0-9, a-Z) 或空白.
& —必填文字，除了空白.
C —必填文字.

- 如果有客製化的需要，可以依 Kendo 的格式建立遮罩，並提供給 rules 的 input 屬性
- 預設會開啟驗證模式(Validation)，可以設置 maskValidation 為 false 關閉
- 可以搭配 Form Validation 一起使用

開關 - Switch

- 設定開/關的文字 [onLabel]="'I'" [offLabel]="'O'"
- 更多其他的設定可以參考 這裡

滑動軸 - Slider

- 設定最大值、最小值、每次移動間格數 [max]="100" [min]="0" [smallStep]="10" [largeStep]="20"
- 預設會顯示遞增/減的按鈕，也可以將其關閉 [showButtons]="false"
- 設定按鈕的標題提示 [incrementTitle]="'增加'" [decrementTitle]="'減少'"
- 預設為水平顯示，也可以控制為垂直顯示 [vertical]="true"
- 更多其他的設定可以參考 這裡
