示例圖

折線圖為例

- 主體大概分為三個部份
- kendo-chart 為主要的 Component，針對 Chart 的主體進行配置
- kendo-chart-title、kendo-chart-legend...等等，為 SubComponent，依功能需求加入
- kendo-chart-series 規劃線圖的呈現方式，如 Line、Bar、Area 等等
- 全部的 Components 可以參考這裡

主體 Kendo-Chart 常用的參數設定

- axisDefaults 可以針對 SubComponent (categoryAxis、valueAxis、xAxis、yAxis) 優先設定預設值
- categoryAxis 針對分類軸線進行設定，比如單位、顏色、最大值、最小值等等
- chartArea 對 Area 類型的 Chart 進行定義
- legend 針對圖例的顯示，設定位置、高度、寬度等等
- seriesDefaults 針對要呈現的數列進行預設值的設定
- series 針對要呈現的數列進行設定，如果某些參數未設定，即會參照 seriesDefaults 的預設值，或是套件本身的預設值
- 更多的設定請參考 這裡

Kendo-Chart 裡常用的元件

- kendo-chart-title 圖表標題的文字設定
- kendo-chart-legend 圖例的顯示設定
- kendo-chart-tooltip 文字提示
- kendo-chart-value-axis 是 kendo-chart-value-axis-item 的集合
- kendo-chart-value-axis-item 針對要呈現的值，進行配置，例如：type = "line" data = "[1, 2, 3]" 等等
- kendo-chart-series kendo-chart-series-item 的集合
- kendo-chart-series-item 針對圖表的呈現設定，外觀、線圖或方塊圖等等
- 更多可以參考這裡

更多說明

因為 Angular 語言特性的關係，Kendo 也不外乎的將每一個 Chart 的功能，拆分成數個 Components，
在使用上，跟傳統的 Jquery 用法上較不相同，透過增加 selector 的方式，組合使用
並透過文件查詢得知有哪些 input 參數可以使用，或是 event 可以觸發。
